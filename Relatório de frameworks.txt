1) O que são Frameworks e para que são utilizados?

Frameworks são conjuntos de ferramentas, bibliotecas e estruturas pré definidas que 
auxiliam no desenvolvimento de aplicações de software. Eles fornecem uma base organizada 
para que os desenvolvedores possam criar sistemas de forma mais rápida, segura e padronizada
evitando a n3ecessidade de implementar funções comuns do zero.
	OBS* No desenvolvimento web, os frameworks são amplamente utilizados para organizar o 
codigo, facilitar a comunicação com o banco de dados, gerenciar rotas, implementar mecanismos 
de segurança  e agilizar a criação de interfaces e funcionalidades. Assim aumentando a produtividade 
e melhorando a praticidade na manutenção do código.



2) Frameworks para Programação Web (Back-end):

	FastAPI (Python) --> O FastAPI é um framework moderno para desenvolvimento de APIs em Python.
Seu principal objetivo é permitir a criação de aplicações web com alto desempenho e pouca quantidade 
de código. Seu diferencial está na velocidade de execução, no suporte nativo á documentação automática 
e na utilização de tipagem Python para validação de dados.

	Javalin (Java) --> O Javalin é um framework leve para desenvolvimento web em java. Seu proposito
é simplificar a criação de aplicações e APIs sem a complexidade presente em frameworks maiores. Seu diferencial 
é a facilidade de aprendizado, a configuração mínima e a leveza da estrutura, tornando-o adequado para projetos 
de pequeno e médio porte.

	CodeIgniter (PHP) -->O CodeIgniter é um framework PHP focado em simplicidade e desempenho. Seu objetivo é 
celerar o desenvolvimento de aplicações web fornecendo ferramentas para manipulação de banco de dados, rotas e formulários.
Seu diferencial é o baixo consumo de recursos e a facilidade de instalação, exigindo poucas configurações para iniciar um projeto.

	Fastify (JavaScript) --> O Fastify é um framework para Node.js voltado para a criação de APIs e aplicações 
web de alta performance. Seu propósito é oferecer uma alternativa mais rápida e eficiente para o desenvolvimento backend em JavaScript.
Seu diferencial é a otimização para processamento de requisições, permitindo excelente desempenho mesmo sob grande volume de acessos.
